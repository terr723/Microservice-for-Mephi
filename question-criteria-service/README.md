# Question Criteria Weight Calculator

Микросервис для расчета весов критериев через семантическое сходство.

## Быстрый старт

```bash
# 1. Установка зависимостей
pip install -r requirements.txt

# 2. Запуск (модель загрузится автоматически)
uvicorn app.main:app --reload

# 3. Откройте в браузере:
http://localhost:8000/docs
